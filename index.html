Feature

スクールの特徴

1.

オタク大歓迎！

アニマギタースクールは、アニメ・漫画オタクさん大歓迎のギター教室です。講師も生徒もみんな「オタク」なので、気兼ね無くオタ語りできます！

好きなアニメソングを好きなだけ弾ける環境でレベルアップを実感してみませんか？もちろんアニソンだけでなく、お好きな曲に挑戦できますよ！

2.

実際にバンドが組める！

アニマでは3ヶ月に1回発表会を開催しています。ライブハウスの利用方法や機材の使い方など、ライブ出演のノウハウを学ぶことができます。

また、ギターレッスンと並行してボーカル、ベース、ドラム、キーボードのレッスンも開講中！ギター以外の楽器も知ることで、音楽に対する理解を深めてみましょう！

3.

経験豊富な講師陣

レッスン動画で有名な海外出身の凄腕講師や、世界で愛されたロックバンドの元ギタリスト、作曲が得意でオリジナル曲は300万再生超えのギタリストなど、実力の高いギタリストが勢揃い。

レッスン内容は生徒さん1人1人に合わせて調整いたしますので、初めてでも安心してレッスンを受けられます。

講師紹介

写真をクリックすると、講師の詳しいプロフィールをご覧いただけます。

ニック 石森
gt/vo/ba/dr

イギリスのマンチェスター出身。小さい頃から日本のアニメが好き。彼のギターレッスン動画は世界中で人気。30歳で念願の日本移住を果たす。

Message

大好きな曲が演奏できる嬉しさを生徒さんに感じてもらうことが僕の喜びです。基本は日本語での指導になりますが、ご希望でしたら英語での指導も受けられますよ。困ったことがあったら気軽に相談してくださいね！

Mizuki
gt/vo/key

18歳でロックバンドJulianeのgt/voとしてデビューし、日本だけでなく世界でライブを開催。2018年の解散を機に、講師業に専念。

Message

ギターを触ったことがない方、ギターを始めたけど挫折してしまった方、色々な奏法を身につけたい方など、どんな方でも大歓迎！生徒さんが楽しんで上達できるよう全力サポートいたします。一緒に頑張りましょう！

武谷 怜央
gt/ba/dr/key

高校時代に制作した曲が動画配信サイトで300万再生以上を記録。アーティストのサポートギタリストを務める傍ら、楽曲制作を続けている。

Message

最初はギターについて何もわからなくて当然です。遠慮なく質問してください！ギターや機材機材選びのお手伝いもできますよ。予算に合わせて最適なものをお勧めします！皆様にお会いできる日を楽しみにしています！
